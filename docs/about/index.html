关于木子

hi，艾瑞巴蒂，我是木子，一名猫奴兼设计师。

新冠疫情之前一直想做一个公众号，想对自己的过往有所沉淀，公众号命名这块儿自己搁置了很久。

偶然读到林清玄《孤独是一个人的清欢》中小千世界的章节，虽对佛教并无了解，但和自己的想法不谋而合，于是借用此词，终于开启了这个公众号的旅程。

这个公众号目前的内容主要包含设计方面的沉淀、译文、读书笔记等，希能对看到文章的童鞋有所帮助。
