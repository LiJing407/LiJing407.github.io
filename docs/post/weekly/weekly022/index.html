2020日本最受欢迎的字体是什么？

https://www.shejipi.com/438573.html

因为疫情原因，由四月推迟的日本字体设计年鉴2020作品展，正在竹尾神保町本店展出中。本次展览由日本字体设计协会举办，从募集的1689件作品中挑选出的优秀作品。奖项分一般部门、学生部门和最佳作品等类别，展览的内容有书籍装帧、标志设计、字体、包装及海报等多种形式。
