🏷 设计新闻

23年后对速度的需求

https://www.underconsideration.com/brandnew/archives/new_master_brand_for_heinz_by_jones_knowles_ritchie.php

研究用户体验的领导者尼尔森-诺曼集团发布的最新的有关桌面端网页速度的文章。摘要：尽管互联网速度有所提高，但网页速度并没有随着时间的推移而提高。

“将页面加载时间缩短哪怕是一秒，都能改善用户的体验，提高转化率。”
