🏷 设计新闻

Atlassian Design System 搬家了

https://atlassian.design/

Atlassian 设计系统是大家最常见到的学习案例，他们的设计系统不仅仅是如何使用他们的标识、品牌色、字体和其他的设计原理，而且还讲述了如何使他们的品牌更加个性闪耀，例如颜色以龙血、牛肉干这样的词去命名。

新的设计系统网站包含以下内容✨：

  1. 纯粹的设计文档网页；
  2. 设计和开发统一目标：用户所需要的内容；
  3. 重新设计和重新构架的基础架构；
  4. 更好的设计开发体验；

还有很多细节优化，enjoy❤️~
