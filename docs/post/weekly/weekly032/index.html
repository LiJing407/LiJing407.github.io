Anima 4.0

https://blog.animaapp.com/introducing-the-1st-design-to-development-platform-41133b24eb7c

Anima 4.0主要功能：

  1. 响应式高保真原型：集成 Figma、Sketch、Adobe XD插件；实时交互原型；响应式布局；
  2. 开发友好：自动生成 React 代码；代码可覆盖
  3. 新的对比模式：透明的叠加方式；减少切换动作；
  4. 协作：通过动态协作功能，展示从设计到开发的工作流程，团队的每个成员都可以更好地参与到项目中。
