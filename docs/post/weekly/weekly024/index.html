🏷 设计灵感

这里有只可爱的蓝猫

https://www.designboom.com/design/blue-cat-milk-packaging-depot-09-03-2020

俄罗斯设计机构Depot为milgrad的bryansk乳品厂开发了新的品牌标识。此次设计目的是在乳品货架上脱颖而出，增加销售，扩大地域和销售渠道，同时展现幽默、善良和甜蜜的特点。在这次包装重新设计中，Depot创造了一只蓝猫，它在看着你哟~
