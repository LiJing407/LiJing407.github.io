🏷 设计文章

开启远程设计冲刺的技巧

https://slack.design/articles/tips-for-running-a-remote-design-sprint/

  1. 充分考虑人们的时间和约束条件⏰。
  2. 精简并分享出你的议程🗓。
  3. 不要跳过破冰的环节🧊。
  4. 为每项工作确定协作工具和排期🛠。
  5. 积极推进项目进程🌟。
  6. 让相关人员知道他们何时以及如何介入🧐。
  7. 灵活计划🤸‍♂️。
