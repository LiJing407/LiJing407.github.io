🏷 设计技巧

双色调风格

https://www.shejipi.com/459132.html

关于双色调风格，其实在国外的一些设计上经常被运用到，比如海报设计、网站和App，这种具有较强的视觉处理，不但能让原本单调的画面变得更时尚，相比普通色彩的图像，更明确表现出特定的氛围。

🏷 设计专访

Tesla Design Studio

https://workwithus.io/interview/tesla

这个位于加州霍桑SpaceX园区飞机库的 Tesla 设计团队，由国际设计师、工程师和模型师组成，在短短十多年内帮助埃隆·马斯克的可持续能源公司颠覆了整个汽车行业。每一个产品，从其全电动车队到充电配件和服装，再到太阳能和电池硬件都从这里开始。

WorkWithUs是一个精选工作机会的网站，包括初创企业、科技公司、创意机构等的招聘机会。

🏷 随便看看

Ada Lovelace as an emoji

https://jennybrennan.com/ada-lovelace-hashflag-emoji/

AdaLovelaceDay 是以计算机科学先驱 AdaLovelace的故事，庆祝女性在STEM领域的成就。

作为一名STEM领域的女性，Jenny Brennan 探讨了设计hashflag的限制因素，以及如何表现 Ada Lovelace 的兼容性和可见性要求。
