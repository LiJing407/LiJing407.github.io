Handz，一款 3D 插画库

https://www.handz.design/

Handz 是一款免费的3D插画库，各种手部手势，支持Figma、XD、Sketch、Photoshop、Illustrator多种格式。

🏷 设计工具

Storybook 6.0

https://medium.com/storybookjs/storybook-controls-ce82af93e430

Storybook是世界上最流行的UI组件工作室。Airbnb、Slack、Lyft、IBM以及数千个行业顶尖团队都在使用它。

这是一个全新的Storybook 6.0插件，它允许你动态地编辑组件的输入，不需要修改代码，用于快速测试组件。
